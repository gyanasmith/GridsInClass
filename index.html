Introduction to CSS Grid

CSS Grid Layout is a powerful tool for creating flexible and responsive web layouts. It allows web developers to design layouts that were previously difficult or impossible to achieve with traditional layout techniques. With CSS Grid, you can create complex web designs with minimal code, giving you more control over how content is displayed on the screen.

CSS Grid Basics

CSS Grid is a two-dimensional layout system, meaning you can control both columns and rows simultaneously. To start using grid, you need to define a grid container and then specify how the items inside the container are positioned in rows and columns.

Example:


    .container {
        display: grid;
        grid-template-columns: 1fr 2fr;
        grid-template-rows: auto;
    }

In this example, the grid container has two columns: the first column takes up 1 fraction of the available space, and the second column takes up 2 fractions.

Key Concepts of CSS Grid

  • Grid Container: The parent element that defines the grid layout. It contains grid items.
  • Grid Item: The child elements of the grid container. Each grid item is placed within a grid cell or spans multiple cells.
  • Grid Lines: The dividing lines that define the boundaries of rows and columns.
  • Grid Tracks: The space between two grid lines (e.g., a row or column).
  • Grid Areas: A rectangular space that spans one or more grid cells.

What are Media Queries?

Media queries are a key component of responsive web design. They allow you to apply specific styles based on conditions such as the size of the viewport, the device type, or the orientation of the screen. By using media queries, you can ensure your website looks great on any device, from mobile phones to desktop computers.

A basic media query might look like this:


@media (max-width: 768px) {
    body {
        background-color: lightblue;
    }
}

Using the 'and' Operator in Media Queries

The 'and' operator in media queries allows you to combine multiple conditions. For example, you can apply styles only when the screen width is below a certain size and the device is in portrait mode.

Here’s an example using the 'and' operator:


@media (max-width: 768px) 
   and (orientation: portrait) {
    body {
        background-color: pink;
    }
}

In this case, the styles will only be applied if both conditions are true: the screen width is 768px or less, and the device is in portrait mode.

Using the 'or' Operator in Media Queries

The 'or' operator in media queries is represented by a comma , and allows you to apply styles if either of the specified conditions is met. This is useful when you want to apply the same styles to multiple scenarios, such as screens below a certain size or screens in a particular orientation.

Here’s an example using the 'or' operator:


@media (max-width: 768px), (orientation: landscape) {
    /* Styles for screens 768px or less OR devices in landscape mode */
    body {
        background-color: lightgreen;
    }
}

In this case, the background color will change to light green if the screen width is 768px or less OR if the device is in landscape mode.
